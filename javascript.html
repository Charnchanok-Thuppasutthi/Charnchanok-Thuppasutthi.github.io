Introduction to JavaScript

JavaScript เป็นภาษาที่ได้รับความนิยมมากที่สุดในโลกเป็นภาษาที่ใช้สำหรับการเขียนเว็บไซต์ โดยในการเขียนเว็บไซต์จะประกอบไปด้วย 3 องค์ประกอบหลักๆคือ
1.HTML ไว้สำหรับเนื้อหาของเว็บไซต์
2.Css ไว้สำหรับการตกแต่งหน้าเว็บไซต์
3.JavaScript ไว้สำหรับควบคุมการกระทำของเว็บไซต์

JavaScript สามารถเปลี่ยนแปลงเนื้อหาของ Html ได้

ยกตัวอย่างเช่น method ที่มีชื่อว่า getElementById()
โดยการทำนคือจะแสดงสิ่งที่อยู๋ใน parameter(id) หากมีการกดปุ่มจะเปลี่ยนข้อมูลที่แสดงเป็นภายในแทน 

                            <p id="demo">JavaScript can change HTML content.</p>

                            <button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>

                            Click Me!
                            </button>

JavaScript can change HTML content.

javascript รองรับทั้ง Double Quote และ Single Quote

จากตัวอย่างข้างบน สามารถเขียนแบบนีไ้ด้
document.getElementById('demo').innerHTML = 'Hello JavaScript';

สามารถเปลี่ยน Attribute ของ Html ได้

โดยใช้ getElementById()เชื่อมกับการคลิ้กปุ่ม

สามารถใช้ JavaScript ในการแก้ไข Css ได้

ใช้ .style.fontSize='...' ในการแก้ไข

JavaScript can change the style of an HTML element.

สามารถใช้ JavaScript ในการซ่อน Html และแสดงได้

โดยการใช้ .style.display='none' เพื่อล่องหน .style.display='block'เพื่อแสดง

I've got magic.

JavaScript Where To

เราจะเขียนJava Script ให้อยู่ใน tag <script>......</script>

Function and Event

ใน javascript เราจะเขียนไว้ใน Function tag <script> และใช้งานโดยใช้คำว่า 'Called'
และเมื่อเรา called function แล้วเราจะเรียกเหตุการณ์ที่เกิดว่า Event

เราสามารถสร้างไฟล์ script ได้ทั้ง head และ body

โดยถ้าหากเราวาง script ไว้ส่วนล่างของ body หรือก็คือไว้ข้างล่าง
จะช่วยเพิ่มความเร็วในการแสดงผลของ browserได้เพราะว่า script ใช้เวลาในการอ่านนาน

ตัวอย่างการกำหนดฟังก์ชั่นที่ head 
                    <head>

                        <script>

                            function Function() {

                                document.getElementById("demo").innerHTML = "Paragraph changed.";

                            }

                        </script>

                    </head>

ตัวอย่างการกำหนดฟังก์ชั่นที่ body

                        <body>

                            <script>

                                function myFunction() {

                                    document.getElementById("demo").innerHTML = "Paragraph changed.";

                                }

                            </script>

                        </body>

External JavaScript

เราสามารถนำไฟล์ JavaScriptเข้ามาใช้ได้
โดยไฟล์จะนามสกุล .js โดยเขียนได้ดังนี้

                            <script src="myScript.js"></script>;
โดยการเขียน .js จะไม่จำเป็นต้องมีการทำ tag <script>......</script>
ข้อดีของการนำไฟล์ .js ไว้ภายนอก Html
1.ทำให้ HTML กับ Code js แยกกันดูง่าย
2.ทำให้ HTML และ javascript อ่านง่าน
3.Cached JavaScript files ช่วยให้โหลดหน้า เว็บเร็วขึ้น

เราสามารถนำ script จากภายนอกมาใช้ได้

External Script ไม่ได้จำกัดที่ว่าต้องมาจากไฟล์ภายในเครื่อง
สามารถมาจากเว็บไซต์อื่นก็ได้ เช่น

   <script src="https://www.w3schools.com/js/myScript1.js"></script></p>
มาจากไฟล์ใน folder อื่น เช่น
   <script src="/js/myScript1.js"></script></p>
มาจากไฟล์อื่น เช่น
   <script src="myScript1.js"></script></p>

JavaScript Output

JavaScript สามารถแสดงผลออกมาได้ในหลายทาง
เขียนใน HTML element เราจะใช้ innerHTML.
เขียนใน the HTML output เราจะใช้ document.write().
เขียนใน an alert box, เราจะใช้ window.alert().
เขียนใน the browser console, เราจะใช้ console.log().

การใช้ innerHTML.

เราจะใช้ document.getElementById(id) โดยที่ id เป็น attribute ไว้กำหนด Html element
โดยการเปลี่ยน 

                            <p id="demo"></p>
                            <script>
                            document.getElementById("demo").innerHTML = 5 + 6;
                            </script>

การใช้ document.write()

                            <!DOCTYPE html>
                            <html>
                                <body>
                                    <script>
                                    document.write(5 + 6);
                                    </script>
                                </body>
                            </html>

ถ้าหากเราใช้ document.write() แล้วไม่ได้เขียนใน tag script หลังจากที่โหลดหน้า Html ขึ้นมาบน browser จะลบ html อื่นๆหมดจากหน้านั้น 

                            <!DOCTYPE html>
                            <html>
                                <body>
                                    <h1>My First Web Page</1>
                                    <p>My first paragraph.</p>
                                    <button type="button" onclick="document.write(5 + 6)">Try it</button>
                                </body>  
                            </html>

การใช้ window.alert()

จะเป็นการการแจ้งเตือนเป็น alert box ของ browser

                            <script>
                                window.alert(5 + 6);
                            </script>

หรือเราสามารถเขียน โดนไม่ต้องมี window ได้ผลลัพธืจะเป็นดังนี้

                            <script>
                                alert(5 + 6);
                            </script>

การใช้ console.log()

คำสั่งนี้จะไม่แสดงบนจอ Html แต่จะแสดงได้เมื่อเกิดการ debug บน console หรือก็คือ กดf12 บนbrowser เลือกช่อง consoleและทำการ run html อีกรอบ

                            <script>
                                console.log(5 + 6);
                            </script>

การใช้ Print

ไว้ใช้สำหรับเรียก print สิ่งนั้นโดยใช้คำสั่ง window.print()

                            <body>
                                <button onclick="window.print()">Print this page</button>>
                            </body>

JavaScript Statement

statement คือประโยคที่มีการกำหนดค่าเช่น

                            var x, y, z;    // Statement 1
                            x = 5;          // Statement 2
                            y = 6;          // Statement 3
                            z = x + y;      // Statement 4

แสดงวาส JavaScript Statement ก็จะมีการกำหนดค่าเหมือนกัน 

                            document.getElementById("demo").innerHTML = "Hello Dolly.";

Semicolons (;)

Semicolons นั้นจะใช้เพื่อกำหนดว่านี่คือจุดหยุดของ statment นี้เช่น 

                                var a, b, c;     // Declare 3 variables
                                a = 5;           // Assign the value 5 to a
                                b = 6;           // Assign the value 6 to b
                                c = a + b;       // Assign the sum of a and b to c

เราสามารถเขียนในบรรทัดเดียวกันได้เลย

                                a = 5; b = 6; c = a + b;

White space

JavaScriptไม่ได้ sensitive ในการเว้นวรรคระหว่างการพิมพ์ แต่การเว้นวรรคทุกค่าจะช่วยให้อ่านง่านขึ้นเช่น

                            var person = "Hege";
                            var person="Hege";
                            var x = y + z;

Line Length and Line Breaks

เช่นกันการขึ้นบรรทัดใหม่ทั้งที่ไม่ได้มี semicolons นั้น JavaScriptจะอ่านไปเรื่อยๆจนกว่าจะเจอ semicolons

                                document.getElementById("demo").innerHTML =
                                "Hello Dolly!";
                                document.getElementById("demo").innerHTML ="Hello Dolly!";

Code Blocks

ในการเขียนฟังก์ชั่นทุกจะมีรูปแบบคือ funtion {........} ทุกอย่างที่อยู่ใน {.....} จะอยู่ในการทำงานของ function นี้ เช่น 

                            function myFunction() {
                                document.getElementById("demo1").innerHTML = "Hello Dolly!";
                                document.getElementById("demo2").innerHTML = "How are you?";
                              }

JavaScript Syntax

Syntax หรือ โครงสร้างของ JavaScript นั้นเป็นดังตัวอย่าง

                            var x, y, z;       // Declare Variables
                            x = 5; y = 6;      // Assign Values
                            z = x + y;         // Compute Values

โดยประเภทของ Value จะแบ่งออกเป็น 2 ชนิดคือ
1.ค่าตายตัว (Fixed values) คือเป็นค่าที่ไม่สามารถเปลี่ยนแปลงได้ เราเรียก Literals
2.ค่าที่ยังไม่ได้กำหนดค่า (Variable values) คือเป็นค่าที่สามารถเปลี่ยนแปลงค่าได้ภายหลัง เราเรียก Variables

Literals ที่สามารถเห็นได้ชัดเจนคือ

1.ตัวเลขที่ถูกเขียนขึ้นมาโดดๆ
2.ข้อความที่ถูกเขียน และครอบด้วย Single quote หรือ Double quote 

                            10.50
                            1001
                            "John Doe"
                            'John Doe'

Variables

ทางโปรแกรมมิ่งนั้น Variables มีไว้สำหรับในการเก็บค่าโดยจะต้องเกิดการประกาศชื่อVariablesก่อนแล้วเก็บ
โดยใน javascript นั้นจะใช้คำสั่ง "var ชื่อ Variables" ในการประกาศตัวแปร
และเครื่องหมาย "เท่ากับ"นั้นมีไว้สำหรับ เก็บข้อมูลเข้า Variables
ดังตัวอย่าง 

                            var x;
                            x = 6;

Operators ตัวดำเนินการ

JavaScript ได้นำดำเนินการคณิตศาสตร์(arithmetic operators ) เช่น + - * / และ ( )วงเล็บ มาใช้ เช่น 

                            (5 + 6) * 10

และ JavaScript ได้นำเครื่องหมาย "เท่ากับ" ซึ่งเป็นตัวดำเนินการแบบ assignment operatorคือไว้สำหรับเก็บค่าข้อมูลเช่น 

                            var x, y;
                            x = 5;
                            y = 6;

Expressions

คือการดำเนินการของ value จาก operation ที่เขียนไว้ใน statement ก่อนหน้านี้แล้ว เช่น

                            5 * 10 ค่าที่จะแสดงคือ50
                            x * 10 สามารถใช้ตัวแปรที่กำหนดมาแล้วมาใช้งานได้
                            "John" + " " + "Doe" หรือการนำตัวอักษรมาต่อกัน

keyword ตัวแปร

keyword คือ ตัวที่ถูกตั้งขึ้นมาเป็นชื่อ เพื่อเก็บค่า โดยถูกตั้งขึ้นมาจากคำสั่ง var

                            var x, y;
                            x = 5 + 6;
                            y = x * 10;

การ comment

ใช้เพื่อให้ผู้เขียนคนต่อๆไปหรือตัวเองเมื่อกลับมาอ่านภายหลัง สามารถเข้าใจได้
โดย syntax เป็นดังนี้ //......
/* ...... */
โดยโค้ดหลังจาก comment เหล่านี้จนจบนั้นจะไม่เกิดการ execute เช่น

                            var x = 5;   // I will be executed
                            // var x = 6;   I will NOT be executed

JavaScript is Case Sensitive

JavaScript การที่ตั้งชื่อตัวแปร แม้ว่าจะมีความหมายเหมือนกันแต่ถ้า เป็นตัวอักษรคนละตัวแบบ
ตัวอักษรภาษาอังกฤษ ตัวเล็ก และ ตัวใหญ่ JavaScriptจะเข้าใจว่า นี่เป็นตัวแปรคนละ

                            var lastname, lastName;
                            lastName = "Doe";
                            lastname = "Peterson";

JavaScript Comment

การ comment อย่างที่กล่าวไว้ก่อนหน้านี้มีไว้สำหรับให้ผู้เขียนคนต่อๆไปหรือตัวเอง เมื่อกลับมาอ่านภายหลัง สามารถเข้าใจได้โดยการอธิบาย
และยังสามารถไว้ใช้สำหรับการทดสอบโค้ด

โดยการ comment นั้นมี 2 แบบ
1.Single line (บรรทัดเดียว)
2.Multi-line (หลายบรรทัด)

Single line

โดย บรรทัดเดียวนั้นเราจะใช้ // โค้ด...........

                                // Change heading:
                                document.getElementById("myH").innerHTML = "My First Page";

                                // Change paragraph:
                                document.getElementById("myP").innerHTML = "My first paragraph.";

                                var x = 5;      // Declare x, give it the value of 5
                                var y = x + 2;  // Declare y, give it the value of x + 2

Multi-line

หลายบรรทัดนั้นจะใช้ /* โค้ด........*/ 

                                /*
                                The code below will change
                                the heading with id = "myH"
                                and the paragraph with id = "myP"
                                in my web page:
                                */
                                document.getElementById("myH").innerHTML = "My First Page";
                                document.getElementById("myP").innerHTML = "My first paragraph.";

JavaScript Variables

Variable คือ ตัวเก็บข้อมูลต่างๆ ยกตัวอย่าง x,y และ z เป็น Variable โดยถูกประกาศจาก var

                        var x = 5;
                        var y = 6;
                        var z = x + y;

หรืออาจจะตั้งให้มีความหมาย หรือ ความยาวกว่านี้ก็ย่อมได้

                        var price1 = 5;
                        var price2 = 6;
                        var total = price1 + price2;

Assignment Operator

เครื่องหมายเท่ากับ "=" ไม่ได้หมายถึงเท่ากับ แบบทางคณิตศาสตร์์แต่เป็นการเท่ากับคือให้ตัวแปรนั้นมีค่า เท่ากับ
ซึ่งหากต้องการการเท่ากับเหมือนในทางคณิตศาสตร์จะต้องใช้เครื่องหมาย "=="

                            x = x + 5

บรรทัดข้างบนการทำงานคือ ให้ x ซ้าย มีค่าเท่ากับ x ตัวก่อน บวกกับ 5
แสดงว่าในการกำหนด statment ผลลัพธ์ต่างๆจะอยู่ทางซ้าย และการทำงานจะอยู่ทางขวา

ประเภทของข้อมูล

ค่าที่เราจะเก็บ มีทั้งเลข อักษร ซึ่งเห็นได้ว่ามันต่างกัน
ข้อมูลเมื่อเราเก็บเข้าไปก็มีการแยกประเภทเหมือนกันเพื่อให้สามารถใช้ในการดำเนินการต่อไปได้
โดยการแยกประเภทนั้นจะแยกจาก Quote ตอนใส่ค่าลง Variables

                        var pi = 3.14; ตัวเลขจะไม่มี quote  
                        var person = "John Doe"; ตัวอักษรจะมีquote ไม่ว่าจะเป็น single quote หรือ double quote
                        var answer = 'Yes I am!';

วิธีการประกาศ

ขั้นตอนการประกาศและใส่ค่านั้นเราสามารถทำได้หลายขั้นตอน

                            var carName; ประกาศตัวแปรก่อน
                            carName = "Volvo"; ใส่ค่าทีหลัง
                            var carName = "Volvo"; ประกาศและใส่ค่าพร้อมกัน
                            var person = "John Doe", carName = "Volvo", price = 200; สามารถประกาศได้พร้อมๆกันหลายๆตัวแต่จะมี "," คั้นระหว่างตัวแปร
                            var person = "John Doe",หรือจะย้ายลงมาบรรทัดใหม่ก็ได้
                            carName = "Volvo",
                            price = 200;

หากประกาศตัวแปรเปล่าไม่มีการใส่ค่า จะกลายเป็นว่าตัวแปรนั้นเป็นประเภท undefined 

                                var carName;

เราสามารถทำให้ตัวแปรที่มีค่ากลายเป็นไม่มีค่า(undefined)ได้โดยการประกาศชื่อซ้ำ

                                var carName = "Volvo";
                                var carName;

เราสามารถประกาศตัวแปรโดยค่าเกิดจากการทำ operation กันอยู่ได้

                                var x = 5 + 2 + 3; ได้ทั้ง number
                                var x = "John" + " " + "Doe"; และ string

ถ้าหากทำการ operate โดยมี quote ผลลัพธ์จะเป็นการต่อ string

                                var x = "5" + 2 + 3;
                                523

หากลองสลับตัว operate ผลลัพธ์จะทำการอ่านเจอ numberก่อนและทำการบวกก่อนแล้วมาต่อstring ทีหลัง

                                var x = 2 + 3 + "5";
                                55

แสดงว่า JavaScript จะอ่านจากซ้ายไปขวา

Dollar Sign $ เครื่องหมาย dollar

การประกาศตัวแปรที่มี Dollar Sign ใน javascript Professional Programmer จะใช้เพื่อเป็นนามแฝงสำหรับ main function ใน JavaScript library.

                            var $$$ = "Hello World";
                            var $ = 2;
                            var $myMoney = 5;

Underscore (_)

การประกาศตัวแปรที่มี Underscoreใน javascript Professional Programmer จะใช้เป็นนามแฝงสำหรับ ตัวแปรที่เฉพาะ (private variable,hidden variable)

                            var _lastName = "Johnson";
                            var _x = 2;
                            var _100 = 5;

JavaScript Operators

ตัวดำเนินการใส่ค่า (assignment)ใช้ตัว "เท่ากับ" (=) เช่น

                            var x = 10;

ตัวดำเนินการใส่ค่า (addition)ใช้ตัว "+" คือการบวกเลขเข้าด้วยกัน เช่น

                            var x = 5;
                            var y = 2;
                            var z = x + y;

ตัวดำเนินการใส่เพิ่มเท่าตัว (multiplication )ใช้ตัว "*" คือการคูณตัวเลขเข้าด้วยกัน เช่น

                            var x = 5;
                            var y = 2;
                            var z = x * y;

การย่อ operator

เราสามารถลดรูปตัวอักษรตอนตั้งสมการได้ ซึ่งจะได้ผลลัพธ์เท่ากับก่อนลดรูป

Operatorตัวอย่างค่าเท่ากับ
+x = yx = y
+=x += yx = x + y
-=x -= yx = x - y
*=x *= yx = x * y
/=x /= yx = x / y
%=x %= yx = x % y
**=x **= yx = x ** y

ตัวอย่างเช่น

                                var x = 10;
                                x += 5;

การรวม string และ number

JavaScript Arithmetic

JavaScript Arithmetic Operators

Arithmetic Operators ตัวดำเนินดชการทางคณิตศาสตร์ นั้นมีไว้ใช้เฉพาะตัวเลข

Operatorตัวอย่างค่าเท่ากับ
+x = yx = y
+=x += yx = x + y
-=x -= yx = x - y
*=x *= yx = x * y
/=x /= yx = x / y
%=x %= yx = x % y
**=x **= yx = x ** y

ตัวอย่างการใช้

เราสามารถใช้กับตัวเลข

                                var x = 100 + 50;

เราสามารถใช้กับตัวแปร

                                var x = a + b;

หรือใช้กับ expression

                                var x = (100 + 50) * a;

องค์ประกอบของสมการ

โดยจะมี 2 ส่วนคือ
1.operands ใช้เรียกกับตัวเลข
2.operator ตัวดำเนินการกับตัวเลข

OperandOperatorOperand
100+50

การเพิ่ม

เราใช้การบวก "+" ในการเพิ่มจำนวน

                                var x = 5;
                                var y = 2;
                                var z = x + y;

การลด

เราใช้การลบ "-" ในการลดจำนวน

                                var x = 5;
                                var y = 2;
                                var z = x - y;

การคูณ

เราใช้การคูณ "*" ในการเพิ่มค่าเป็นเท่าๆ

                                var x = 5;
                                var y = 2;
                                var z = x * y;

การหาร

เราใช้การหาร "/" ในการหาร

                                var x = 5;
                                var y = 2;
                                var z = x / y;

การหาเศษ

เราใช้การม้อด(modulus) "%" ในการหาเศษ

                                var x = 5;
                                var y = 2;
                                var z = x % y;

การเพิ่ม(Incrementing)

เราใช้เครื่องหมาย "++" ในการหาเพิ่มจำนวนไป 1 

                                var x = 5;
                                x++;
                                var z = x;

การเพิ่ม(Decrementing)

เราใช้เครื่องหมาย "--" ในการหาลดจำนวนไป 1 

                                var x = 5;
                                x--;
                                var z = x;

การยกกำลัง(Exponentiation)

เราใช้เครื่องหมาย "**" ในการยกกำลัง 

                                var x = 5;
                                var z = x ** 2;

เราสามารถใช้คำสั่ง "Math.pow(x,y):" หายกกำลังได้

                                var x = 5;
                                var z = Math.pow(x,2);   // result is 25

ลำดับความสำคัญ Precedence

ตัวอย่าง หากสมการมีการคำนวณมากกว่า 1 operator แต่ละ arithmetic operator
จะมีลำดับความสำคัญในการดำเนินการอยู่คือจะมีลำดับตอนเริ่มคำนวณก่อน-หลัง
โดยจะเริ่มจาก การคูณ "*" การหาร "/" การบวก "+" การลบ "-"
โดยเราสามารถใช้ วงเล็บ () ในการกำหนด ลำดับความสำคัญในการดำเนินการได้ เช่น

                                var x = 100 + 50 * 3;   //250
                                var x = (100 + 50) * 3; //450
