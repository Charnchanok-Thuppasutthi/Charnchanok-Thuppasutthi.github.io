ในหัวข้อ < HTML Elements >

คำว่า Elements จะนิยามถึงทุกอย่างที่เริ่มจาก tag เปิดจนถึงปิด โดย มี tag

Elements สามารถติดกันได้ คือสามารถเรียกใช้หลายๆ tag พร้อมกันได้

ยกตัวอย่างCode

                <!DOCTYPE html> 

                <html> 

                  <body> 


                    <h1> My First Heading</h1> 

                    <p> My first paragraph.</p> 


                  </body> 

                </html>

จะเห็นได้ว่า <html><body><h1> สามารถติดกันได้
ส่วน code ที่อยู่ใน <html> จนถึงฝั่งปิด(</html>) จะเป็นส่วนทั้งหมดของ file HTML นั้นๆ

หรือก็คือส่วน 
                  <body> 

                    <h1> My First Heading</h1> 

                    <p> My first paragraph.</p> 

                  </body>

ส่วนข้างในของตัว <body> นี้คือ code ที่อยู่ใน body 

                  <h1> My First Heading</h1> 

                  <p> My first paragraph.</p>

ซึ่ง <h1>....</h1> ไว้สำหรับเขียน heading1 (หัวข้อที่1)
การที่มี <h1>....</h1> การมีเลข 1 เท่ากับว่าสามารถมี <h2></h2> , <h3></h3>...ถึง <h6></h6>
ซึ่ง <p>....<p> ไว้สำหรับเขียน paragraph
<br> ไว้สำหรับเริ่มไปบรรทัดใหม่

พิมพ์ใหญ่ กับ พิมพ์เล็ก จะไม่ใช่คำสั่งเดียวกันใน HTML

เช่น
<P> ไม่ได้มีการทำงานเหมือนกับ <p>

<HTML Attribute>

<a> เป็น tag ที่ใช้สำหรับกำหนด hyperlink href=..... ใส่เป็น URL

<a href="https://www.w3schools.com">Visit W3Schools</a>
Visit W3Schools

src Attribute เราใช้ tag นี้ไว้ฝังภาพลงไปบนเว็บไซต์ เช่น

<img src="img_girl.jpg">

width และ height Attribute ไว้สำหรับ

<img src="img_girl."jpg width="500" height="600">

alt Attribute เป็น attribute สำหรับ tag <img> ใช้เพื่อแสดงประโยคเพื่อบ่งบอกว่าภาพนี้เป็นภาพอะไรในกรณีที่ ไม่สามารถแสดงภาพได้จากสาเหตุด้าน internet

<img src="img_typo.jpg" alt="Girl with a jacket">

style Attribute ใช้สำหรับเพิ่ม color font size และอื่นๆให้กับประโยค

<p style="color:red;">This is a red paragraph.</p>

This is a red paragraph.

ในหัวข้อ < HTML Heading >

Heading หมานถึวหัวข้อ ข้อความการประกาศ Heading นี้จะใข้ Tag <h1>, <h2> ,<h3> ไปจนถึง 6 เช่นตัวอย่างนี้

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

โดยเรียงลำดับความสำคัญ จากเลขจ้อยไปมาก

โดยปกติมักจะใช้ <h1> เป็นหัวข้อหลัก <h2> หัวข้อรองลงมา <h3>ก็รองลงมา เรื่อยๆอีก

หากต้องการเพิ่มขนาดของ Heading เราจะใช้ attribute ของ option style ที่ชื่อว่า font-size

<h1> style="font-size:60px;">Heading 1 </h1>

ในหัวข้อ < HTML Paragraphs >

ใน HTML เราใช้คำ Tag <p> เพื่อทำการกำหนดว่า นี่คือประโยค โดยทุกครั้งที่ใช้งาน Tag จะทำการขึ้นบรรทัดใหม่ เช่น

<p> This is a paragraph. </p>
<p> This is another paragraph. </p>

เมื่อเปิดผ่าน Browser จะได้ผลลัพธ์ได้ดังนี้

This is a paragraph.

This is another paragraph.

เราไม่สามารถทราบได้ถึงขนาดของหน้าจอ(Display)ของผู้เข้าดูได้แต่ HTML นั้นจะทำการจัดการพื้นที่ของข้อความให้อัตโนมัติ

เช่น <p> This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

การแสดงผลลัพธ์จะเห็นดังข้างล่าง

This paragraph contains a lot of lines in the source code, but the browser ignores it.

Html Css

ซึ่ง Css นั้นย่อมาจาก Cascading Style Sheets มีไว้สำหรับจัดโครงสร้างของ webpage
โดย CSS เราได้ทำการเขียนไปแล้วบ้างในหัวข้อก่อนโดยที่เราไม่รู้ตัว
โดย CSS นั้นการเขียนมีทั้งหด 3 แบบ
Inline คือใช้ในบรรทัด ซึ่งที่ผ่านมาเราได้ใช้ตัวนี้มาพอสมควร
Internal จะประกาศใช้ใน1ส่วนของโครง Html เช่น <Head>...ถึง...</Head>
External จะประกาศใช้จากไฟล์ภายนอก Html

Inline Css เช่น
<h1 style="color:blue;">A Blue Heading</h1>

Internal Css ใช้กับ Html ในไฟล์นั้นๆ หรือ 1 Page นั่นเอง
โดยจะกำหนดที่ส่วนของ <Head> ใน <style>


<head>
<style>
body {background-color: powderblue;}
h1 {color: blue;}
p {color: red;}
</style>
</head>

External Css คือการกำหนดผ่านไฟล์ภายนอก ใช้กับ Html หลายๆไฟล์ได้

<head>
<link rel="stylesheet" href="styles.css">
</head>

นี่คือการเรียกใช้ ไฟล์ css
body {
background-color: powderblue;
}

color: blue;
}

color: red;
}

HTML Links

Link เปรียบเสมือนทาง เพือ่ไปถึงจุดหมายปลายทาง(Direction)ในเว็บไซต์ โดย
Links ใน Html นั้นจะเป็น Hyperlink คือคลิ้กแล้วไปหน้า page อื่นๆ

ไวยากรณ์ คือ >a<โดยเป็นการกำหนดว่านี่คือ tag link นะ

<a href="url">link text</a>
<href> คือการกำหนด address ของ link

Target Attribute ไว้สำหรับกำหนดการกระทำเมื่อกดลิ้งว่าจะเปิด หน้าใหม่หรือเปลี่ยนหน้าปัจจุบัน

โดยจะมีดังนี้
_self - เป็น default จะเปลี่ยนหน้าปัจจุบัน
_blank - เปิดหน้าใหม่
_parent - Opens the document in the parent frame
_top - Opens the document in the full body of the window

Absolute URLs คือเว็บภายนอก
Relative URLs คือเว็บภายในเว็บไซต์เดียวกับที่เราอยู่
