ทำการอธิบายที่ได้ศึกษา HTML

ในหัวข้อ < HTML Elements >

คำว่า Elements จะนิยามถึงทุกอย่างที่เริ่มจาก tag เปิดจนถึงปิด โดย มี tag

Elements สามารถติดกันได้ คือสามารถเรียกใช้หลายๆ tag พร้อมกันได้

ยกตัวอย่างCode

<!DOCTYPE html>
<html>
<body>

<h1> My First Heading</h1>
<p> My first paragraph.</p>

</body>
</html>

จะเห็นได้ว่า <html><body><h1> สามารถติดกันได้
ส่วน code ที่อยู่ใน <html> จนถึงฝั่งปิด(</html>) จะเป็นส่วนทั้งหมดของ file HTML นั้นๆ หรือก็คือส่วน

<body>
<h1> My First Heading</h1>
<p> My first paragraph.</p>
</body>

ส่วนข้างในของตัว <body> นี้คือ code ที่อยู่ใน body

<h1> My First Heading</h1>
<p> My first paragraph.</p>

ซึ่ง <h1>....</h1> ไว้สำหรับเขียน heading1 (หัวข้อที่1)
การที่มี <h1>....</h1> การมีเลข 1 เท่ากับว่าสามารถมี <h2></h2> , <h3></h3>...ถึง <h6></h6>
ซึ่ง <p>....<p> ไว้สำหรับเขียน paragraph
<br> ไว้สำหรับเริ่มไปบรรทัดใหม่

พิมพ์ใหญ่ กับ พิมพ์เล็ก จะไม่ใช่คำสั่งเดียวกันใน HTML

เช่น
<P> ไม่ได้มีการทำงานเหมือนกับ <p>

<HTML Attribute>

<a> เป็น tag ที่ใช้สำหรับกำหนด hyperlink href=..... ใส่เป็น URL

<a href="https://www.w3schools.com">Visit W3Schools</a>
Visit W3Schools

src Attribute เราใช้ tag นี้ไว้ฝังภาพลงไปบนเว็บไซต์ เช่น

<img src="img_girl.jpg">

width และ height Attribute ไว้สำหรับ

<img src="img_girl."jpg width="500" height="600">

alt Attribute เป็น attribute สำหรับ tag <img> ใช้เพื่อแสดงประโยคเพื่อบ่งบอกว่าภาพนี้เป็นภาพอะไรในกรณีที่ ไม่สามารถแสดงภาพได้จากสาเหตุด้าน internet

<img src="img_typo.jpg" alt="Girl with a jacket">

style Attribute ใช้สำหรับเพิ่ม color font size และอื่นๆให้กับประโยค

<p style="color:red;">This is a red paragraph.</p>

This is a red paragraph.

ในหัวข้อ < HTML Heading >

Heading หมานถึวหัวข้อ ข้อความการประกาศ Heading นี้จะใข้ Tag <h1>, <h2> ,<h3> ไปจนถึง 6 เช่นตัวอย่างนี้

Heading 1

Heading 2

Heading 3

Heading 4

Heading 5
Heading 6

โดยเรียงลำดับความสำคัญ จากเลขจ้อยไปมาก

โดยปกติมักจะใช้ <h1> เป็นหัวข้อหลัก <h2> หัวข้อรองลงมา <h3>ก็รองลงมา เรื่อยๆอีก

หากต้องการเพิ่มขนาดของ Heading เราจะใช้ attribute ของ option style ที่ชื่อว่า font-size

<h1> style="font-size:60px;">Heading 1 </h1>
ผลลัพธ์ที่ได้จะเป็นดังข้างล่าง

Heading 1

ในหัวข้อ < HTML Paragraphs >

ใน HTML เราใช้คำ Tag <p> เพื่อทำการกำหนดว่า นี่คือประโยค โดยทุกครั้งที่ใช้งาน Tag จะทำการขึ้นบรรทัดใหม่ เช่น

<p> This is a paragraph. </p>
<p> This is another paragraph. </p>

เมื่อเปิดผ่าน Browser จะได้ผลลัพธ์ได้ดังนี้

This is a paragraph.

This is another paragraph.

เราไม่สามารถทราบได้ถึงขนาดของหน้าจอ(Display)ของผู้เข้าดูได้แต่ HTML นั้นจะทำการจัดการพื้นที่ของข้อความให้อัตโนมัติ

เช่น <p> This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

การแสดงผลลัพธ์จะเห็นดังข้างล่าง

This paragraph contains a lot of lines in the source code, but the browser ignores it.

Tag <hr>ไว้สำหรับขีดเส้นใต้หน้าใน HTML

เช่นเส้นใต้นี้

เป็นต้น

Tag <br> (Line Break) ไว้สำหรับขึ้นบรรทัดใหม่

เช่น
<p>This is<br>a paragraph<br>with line breaks.</p>

ผลลัพธ์ที่ได้คือ

This is
a paragraph
with line breaks.

Tag <pre> ใช้สำหรับการแสดงข้อความที่จะออกมาเป็นเหมือนที่เราพิมพ์ใน HTML

ถ้าเราทำการพิมพ์

<pre> My Bonnie lies over the ocean.

My Bonnie lies over the sea.

My Bonnie lies over the ocean.

Oh, bring back my Bonnie to me.
</pre>

ผลลัพธ์ที่ได้จะเป็นเหมือนที่พิมพ์ไป

      My Bonnie lies over the ocean.
    
      My Bonnie lies over the sea.
    
      My Bonnie lies over the ocean.
    
      Oh, bring back my Bonnie to me.

ในหัวข้อ < HTML Style >

style เป็น attribute ที่ใช้สำหรับเพิ่มelement(สี,ฟ้อนต์,ขนาดและอื่นๆ)

เช่น

I am normal

I am red

I am blue

I am big

จากโค้ด

I am normal
<p style="color:red;">I am red </p>
<p style="color:blue;">I am blue </p>
<p style="font-size:50px;">I am big </p>

syntax หรือ ไวยากรณ์ของ style จะเป็นดังโครงสร้างนี้

< tagname > style="property:value;" </tagname>

รูปแบบฟ้อนต์

This is a heading

This is a paragraph.

Centered Heading

Centered paragraph.

จากโค้ด

<p style="text-align:center;">Centered paragraph.</p>

ในหัวข้อ < HTML Formatting >

ลักษณะพิเศษที่สามารถใช้งานกับ ตัวอักษร มีดังนี้

<b> - Bold text
<strong> - Important text
<i> - Italic text
<em> - Emphasized text
<mark> - Marked text
<small> - Smaller text
<del> - Deleted text
<ins> - Inserted text
<sub> - Subscript text
<sup> - Superscript text

<b>และ<strong> ทำให้ตัวอักษรหน้าเหมือนกัน อย่างตัวอย่างข้างล่าง

This text is bold

<i>และ<em> ทำให้ตัวอักษรเอียง อย่างตัวอย่างข้างล่าง

This text is italic

<small> ทำให้ตัวอักษรเล็กกว่าปกติ

This is some smaller text.

<mark>ทำการ Highlight ตัวอักษรนั้น

Do not forget to buy milk today.

<del>ทำการขีดค่าตัวอักษรนั้น

My favorite color is blue red.

<ins>ทำการขีดเส้นใต้ตัวอักษรนั้น

My favorite color is blue red.

<sub>ทำให้ตัวอักษรเป็นตัวไขว้ใต้ตัวปกติ

This is subscripted text.

<sub>ทำให้ตัวอักษรเป็นตัวไขว้ใต้ตัวปกติ

This is superscripted text.
